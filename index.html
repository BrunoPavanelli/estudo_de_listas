Lista Ordenada

Lista Ordenadas seguem uma ordem especifica de precedencia podendo ser usadas para indicar ações que possuem passos em ordem como receitas, instalações de software, etc...

  1. a
  2. b
  3. c

Lista Não-Ordenada

Lista Não-Ordenadas já não possuem ordem de precedência, podendo ser usadas de forma mais abrangente e genérica:

  • a
  • b
  • c

Lista de Descrição

Lista de Descrição servem para listar elementos e explica-los em ordem, como no exemplo abaixo:

(Description List)

Tags Inline e Block

Tags em Linha (Inline)
Tags que ocupam o o espaço de largura necessário para sua visualização
Tags em Bloco (Block)
